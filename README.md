verb-ios
========
Verb iOS app written in Swift.

#### Environment Config
Create `scripts/config.sh` based on sample in `scripts/config.sh.sample`

#### Build from Command Line
`./scripts/run-locally.sh`

#### Updating Encrypted Certs/Profiles
`./scripts/encrypt.sh`
